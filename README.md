# First-pytest-project

This repository was an exercise project for learning pytest. It contains example tests, configuration files, and generated HTML/Allure reports to demonstrate common testing workflows.

## Contents

* `Tests/` — test cases and test modules.
* `Utilities/` — helper modules used by tests.
* `conftest.py` — pytest fixtures and global test setup.
* `pytest.ini` — pytest configuration (markers, test paths, addopts, etc.).
* `AllureReport/`, `AutomationPytestReport.html`, `AutomationPageObjectReport.html` — example test reports.
